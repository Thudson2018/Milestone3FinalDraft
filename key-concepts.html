Key Concepts for Browsers

What is a Browser: A browser is essentially a portal connecting users across the world to the internet where we store a vast array of information, forms, and others tools for others. With this very wide topic alot of terms and ideas will be said throughout this, and this page will be the intro into having a better concept of what this website it for.

Rendering Engine

The "Rendering Engine" is how the website, or browser communicates with the Operating System (OS) to complete and compile requests for dispalying differnt things on a user's screen. Think of it as the brain and blood of the computer. The OS is responisble for almost every function of a computer either voluntary or involuntary.

Popular Rendering Engines

  • UI: How a user can interact and use a browser or website
  • Rendering: The process of compiling code within pages and displaying it
  • Networks : For the management in processing HTTP, and HTTPS requests

HTTP and HTTPS Protocols

HyperText Transfer Protocol (HTTP) and HyperText Transfer Protocol Secure(HTTPS) are two protocols that our browser and network are responsible for in the transfering of data. HTTP is the more basic process used in trasfering smaller less important bits of data. HTTPS encrypts the HTTP file increasing the security of the user attempting to share data and do it safely.

Cookies and Cache

Cookies are bits of data that your computer will store to make the overall experience better for a user. These bits of data could be something like login passwords & user's, or a shopping cart your working on. It is a more user focused version of a cashe, which is focused more on the overall speed and efficency of the browser. Within the cashe a browser will store thinkgs like the next page of a website, images or files that have a possiblity of being opened in the near future.

Browser Standards

Due to the amount of browsers out there, it is important for everyone to have a standard they have to adhere too.

  • World Wide Web Consortium: Standard for accessibility
  • WHATWG: Maintaince the HTML standards
  • Internet Engineering Task Force: Sets HTTP, and HTTPS protocol
  • ECMA International: Standarizes use of JavaScript

Best Practice

Always keep your browser updated to the latest version. Browser updates often include critical security patches that protect you from newly discovered vulnerabilities.
